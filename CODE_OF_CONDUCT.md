# Code of Conduct

## Expectations

Participation in this project requires respectful, professional behavior.

- Be direct about technical issues without making discussions personal.
- Assume good intent and ask for clarification before escalating.
- Keep feedback constructive, specific, and relevant to the work.
- Respect privacy and avoid sharing secrets, personal data, or security-sensitive details publicly.

## Unacceptable Behavior

- Harassment, discrimination, threats, or intimidation
- Personal attacks, insults, or deliberate hostility
- Doxxing, stalking, or publishing private information
- Repeated disruption of project discussions or reviews

## Scope

This code of conduct applies to project spaces, including issues, pull requests, code review, and other repository-hosted discussions.

## Enforcement

Maintainers may edit or remove content, pause discussions, reject contributions, or block participation when behavior violates this policy.

## Reporting

Report conduct concerns privately to the maintainers through the repository hosting platform when possible. Public escalation should avoid sharing sensitive personal information.
